- **Connect**: comunicazioni medico-paziente (senza lo strumento cartella clinica, non si possono elaborare le ricette).
- **Smart**: solo strumento cartella clinica, si possono elaborare le ricette ma non si può gestire la comunicazione medico-paziente
- **Premium**: tutto incluso.

| **Funzionalità**                                                                | **Connect** | **Smart** | **Premium** |
| ------------------------------------------------------------------------------- | ----------- | --------- | ----------- |
| Agenda con prenotazione via app                                                 | Sì          | No        | Sì          |
| Triage referti con risposte pre-compilate condivisibili via app                 | Sì          | No        | Sì          |
| Campagne di avvisi per gruppi di pazienti (età, sesso e patologie)              | Sì          | No        | Sì          |
| Chat con opzione spegnimento nei weekend e/o a necessità                        | Sì          | No        | Sì          |
| Videovisita                                                                     | Sì          | No        | Sì          |
| Richieste di ricette via app                                                    | Sì          | Sì        | Sì          |
| Cartella Clinica Elettronica in cloud                                           | No          | Sì        | Sì          |
| Prescrizioni (SSN e non) con notifica al paziente                               | No          | Sì        | Sì          |
| Rinnovo massivo prescrizioni con calcolatore posologico                         | No          | Sì        | Sì          |
| Allineamento automatico anagrafiche ed esenzioni                                | No          | Sì        | Sì          |
| Trascrizione automatica dei referti PDF o immagini con Intelligenza Artificiale | No          | Sì        | Sì          |

Il medico non può modificare metodo di pagamento o disdire l’abbonamento in autonomia.

- Se serve modificare il metodo di pagamento o il rinnovo del pagamento non è andato a buon fine, serve che ci indichi una fascia oraria in cui il medico è disponibile così da essere contattato telefonicamente
- Se deve disdire l’abbonamento, deve inviarci una PEC all’indirizzo di posta elettronica: [davincihealth@pec.it](mailto:davincihealth@pec.it)
